# Santa Monica College EEO IBP Dashboard

Interactive project management dashboard for Santa Monica College's Equal Employment Opportunity Innovative Best Practices grant (2026–2028).

## Overview

This dashboard provides a comprehensive view of SMC's EEO IBP grant initiative, including:

- **Grant Details**: $150,000 funding over 30 months (January 2026 – June 2028)
- **Focus Areas**: Employee Resource Groups (ERGs), Mentorship Programs, Professional Development, Conference Participation
- **Interactive Navigation**: Easy access to Goals, Budget, Timeline, and Team Contacts
- **SMC Branding**: Official Santa Monica College colors and design standards

## Features

- **Overview Section**: Key metrics and grant summary
- **Goals & Objectives**: Detailed program goals and measurable outcomes
- **Budget Breakdown**: Complete 2-year budget allocation across all categories
- **Timeline**: Implementation schedule from planning through completion
- **Contact Information**: Full project team with roles and responsibilities
- **Dark/Light Mode**: Theme toggle for user preference
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Technology

- Pure HTML/CSS/JavaScript (no build process required)
- Google Fonts (Crimson Text & Source Sans 3)
- Chart.js for future data visualization
- CSS Grid and Flexbox for responsive layouts
- CSS Custom Properties for theming

## Live Site

View the dashboard at: **https://wardere83.github.io/smcdashboard/**

## Data Source

All grant information sourced directly from the Santa Monica College EEO Innovative Best Practices application submitted for the 2026–2028 funding cycle.
